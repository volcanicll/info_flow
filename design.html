GPT-5 正式发布：多模态推理、256K 上下文与全新 Agent 能力

38 分钟前36kr.com

OpenAI 在今日凌晨的开发者直播中正式发布 GPT-5，将 o 系列的推理能力与 4o 的实时多模态整合进单一模型，并把上下文窗口提升至 256K。

新模型在 SWE-Bench Verified 上达到 71.2%，相较 GPT-4o 提升约 19 个百分点；在 AIME 数学基准上的表现也首次追平了专为推理优化的 o3。API 方面，输入定价为每百万 token 2.74 美元，较 4o 下调约 41%。

统一的推理与多模态

GPT-5 取消了显式的"思考模式"开关，改为模型自主决定推理深度。开发者可通过 reasoning_effort 参数在 latency 与质量之间权衡，最低档位下首 token 延迟控制在 410ms。

多模态方面，模型可直接接收图像、音频与短视频片段，并支持在单次会话中跨模态引用。现场演示中，工程师对着白板照片追问代码实现，模型给出了可直接运行的 TypeScript 版本。

Agent 能力与定价

伴随 GPT-5，OpenAI 推出了新的 Codex CLI 与浏览器 Agent 工具包，允许模型在沙箱中执行命令、读写文件并维护长期任务状态。免费用户每周可获得 40 次调用，Plus 与 Team 套餐即刻可用。

AI 智能摘要

OpenAI 发布 GPT-5，统一推理与多模态，上下文扩至 256K。SWE-Bench 达 71.2%，AIME 追平 o3。API 输入定价下调约 41%，并推出 Codex CLI 与浏览器 Agent，免费用户每周 40 次调用。
